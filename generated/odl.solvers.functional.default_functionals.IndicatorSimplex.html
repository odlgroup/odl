Examples

Example where a point lies outside the unit simplex ...

>>> space = odl.rn(3)
>>> ind_simplex = IndicatorSimplex(space)
>>> x = space.one()
>>> ind_simplex(x)
inf

... and one where it lies inside the unit simplex.

>>> x /= x.ufuncs.sum()
>>> ind_simplex(x)
0
