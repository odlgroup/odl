ImagPart¶

Examples

Take the imaginary part of complex vector:

>>> c3 = odl.cn(3)
>>> op = ImagPart(c3)
>>> op([1 + 2j, 2, 3 - 1j])
rn(3).element([ 2.,  0., -1.])

The operator is the zero operator on real spaces:

>>> r3 = odl.rn(3)
>>> op = ImagPart(r3)
>>> op([1, 2, 3])
rn(3).element([ 0.,  0.,  0.])
