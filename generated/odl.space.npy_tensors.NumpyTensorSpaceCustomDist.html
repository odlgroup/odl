NumpyTensorSpaceCustomDist¶

class odl.space.npy_tensors.NumpyTensorSpaceCustomDist(dist)[source]¶

Bases: CustomDist

Class for handling a user-specified distance in TensorSpace.

Note that this removes inner and norm.

Initialize a new instance.

Parameters:
distcallable

The distance function defining a metric on TensorSpace. It must accept two Tensor arguments, return a float and fulfill the following mathematical conditions for any three elements x, y, z:

  • dist(x, y) >= 0

  • dist(x, y) = 0 if and only if x = y

  • dist(x, y) = dist(y, x)

  • dist(x, y) <= dist(x, z) + dist(z, y)
