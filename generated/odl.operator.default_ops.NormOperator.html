NormOperator¶

class odl.operator.default_ops.NormOperator(*args, **kwargs)[source]¶

Bases: Operator

Vector space norm as an operator.

Implements:

NormOperator()(x) <==> x.norm()

This is only applicable in normed spaces, i.e., spaces implementing a norm method.

See also
