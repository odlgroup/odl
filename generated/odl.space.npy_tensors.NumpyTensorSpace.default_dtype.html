NumpyTensorSpace.default_dtype¶

static NumpyTensorSpace.default_dtype(field=None)[source]¶

Return the default data type of this class for a given field.

Parameters
fieldField, optional

Set of numbers to be represented by a data type. Currently supported : RealNumbers, ComplexNumbers The default None means RealNumbers

Returns
dtypenumpy.dtype

Numpy data type specifier. The returned defaults are:

RealNumbers() : np.dtype('float64')

ComplexNumbers() : np.dtype('complex128')
