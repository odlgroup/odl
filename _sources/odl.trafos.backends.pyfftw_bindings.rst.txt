pyfftw_bindings
===============

Bindings to the ``pyFFTW`` back-end for Fourier transforms.

The `pyFFTW <https://pyfftw.readthedocs.io>`_  package is a Python
wrapper around the well-known `FFTW <http://fftw.org/>`_ library for fast
Fourier transforms.
